# dot-config

## Usage

This repository tracks my personal configuration, and provides an easy way to bootstrap this config on an Ubutnu system.


## Install

`install/` contains scripts for installing individual or very small sets of packages.

## Dot Configs

### ~/.gitconfig

Requires user and email replacement.

There is a dependency on `delta` a colored diff tool for git. This .deb must be downloaded from releases page.

### ~./zshrc 

Requries `GIT_PAT` environment variable.
